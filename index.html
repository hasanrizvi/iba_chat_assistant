An AI Powered Chatbot For IBA's Website As A Browser Extension

Summary

This is an AI-powered chatbot encapsulated within a browser extension, strategically employing machine learning methodologies to proficiently address inquiries pertaining to Institute of Business Administration (IBA). The extension transforms the browsing experience by delivering prompt and contextually informed responses, thereby optimizing access to IBA-related information. The machine learning algorithms embedded within the extension provide users with an interactive support system for navigating the complexities of IBA's website.

Novelty

The chatbot application provides a user-friendly interface that greets users upon activation and offers options for assistance. It effectively processes natural language queries through API requests to a backend server, which retrieves information from a vector database covering admissions procedures, academic programs, campus facilities, and FAQs. Responses are formulated based on the search results and may include links to relevant university website pages. The interface prioritizes usability with a clear, minimalistic design to ensure ease of interaction. Security measures include secure data transmission via REST APIs and no storage of user data on the backend server, thus minimizing the risk of data breaches. Performance goals focus on speed, reliability, and scalability to maintain a seamless user experience with quick response times and the ability to handle concurrent requests effectively.

Design Diagram

This is a design diagram which represents how this application works. First of all a user has to ask a query to start the process. The query is sent to the backend server for processing When the query reaches the backend server it is matched with the closet document present in the database. The language model then extracts the relevant information and formulates a message. The message is sent to the frontend application and it is shown as a chat message to the user.
